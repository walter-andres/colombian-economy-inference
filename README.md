# Colombian Economy — Inferential Statistics Case Studies

This repository contains a collection of applied case studies focused on the Colombian economy using inferential statistics.

The objective is to practice statistical reasoning, hypothesis testing, confidence intervals, regression inference, multilevel models, and marginal models through real or simulated economic data related to Colombia.

The study cases are designed to connect statistical theory with practical questions that can be answered using data from Colombian institutions such as DANE, Banco de la República, and other official sources.

---

## Project Objective

The main purpose of this repository is to develop inferential statistics exercises applied to the Colombian economy.

Each notebook focuses on a specific statistical method and uses an economic question as the basis for analysis.

The repository is intended to help understand how inferential statistics can be used to:

- Compare groups
- Estimate population parameters
- Test hypotheses
- Analyze relationships between variables
- Model economic outcomes
- Interpret statistical evidence
- Support data-driven conclusions

---

This repository includes study cases related to the following inferential statistics methods:
